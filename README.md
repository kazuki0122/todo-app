# todo-app

todo-app はタスク管理アプリです。あなたの PDCA サイクルを回すためのサポートをします。  
ジャンル毎にタスクを決め、日々の達成率を確認できます。  
また、一週間あたりのジャンル毎の達成率も確認できます。

# 使用技術

- 使用技術/フレームワーク

  - フロントエンド: Vue.js
  - バックエンド: Firebase

- Vue.js ライブラリ
  - Vuetify
  - Vuex
  - Vue Router
  - vuedraggable

# 機能

1. ログイン・ログアウト機能
2. タスクのソート機能
3. ジャンル・タスクの CRUD 機能
4. 一週間あたりの進捗度表示機能
